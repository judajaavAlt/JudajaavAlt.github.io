Archivos:
    informe.pdf
        En este archivo se encuentra toda la sustentacion teorica del proyecto junto a las justificaciones y conclusiones.
    icon.svg
        Este archivo sirve como icono de la pagina web
    index.html
        Este archivo contiene la estructura de la pagina web donde se podran ejecutar las pruebas.
        consta de un formulario a forma de input, y un output donde se puede comparar el input con el output
    style.css
        Este archivo contiene los estilos necesarios para que la pagina se vea bien y sea comoda
    main.js
        Aqui se encuentran los algoritmos modexFB, modexPD, modexV. ademas contiene una funcion que parsea los documentos, los ejecuta y retorna los resultaods
    README.txt
        es este archivo, la finalidad de este es informar sobre el contenido de la carpeta

Como ejecutar?:
    teniendo node.js
        1. debe ubicarse en la carpeta del documento y abrir una consola
        2. corra el siguiente codigo "npx http-server -o"

    sin node.js
        como alternativa se puede acceder de manera online al archivo en la siguiente direccion: https://judajaavalt.github.io/
